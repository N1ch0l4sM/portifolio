Airline Analysis

In this project, I conducted an in-depth analysis of airline flight data. The dataset included variables such as miles flown, number of passengers, delay times, coach and first-class prices, and flight duration. The primary tool used for this analysis was Python, with libraries including pandas for data manipulation, seaborn for data visualization, and matplotlib for additional plotting features. One of the key components of the project was the creation of a heatmap to visualize the correlations between the different variables in the dataset. This provided valuable insights into how these variables interact with each other. For instance, it helped identify if longer flights tend to have more delays, or if flights with higher first-class prices tend to carry more passengers. The heatmap was carefully crafted with seaborn and matplotlib, ensuring optimal color selection for clear visibility of correlations and appropriate rotation of labels for better readability. This project showcased my ability to manipulate data using pandas, create meaningful visualizations with seaborn and matplotlib, and draw insights from complex datasets.

Cleaning US Census Data

In this project, I undertook the task of cleaning and analyzing a dataset from the US Census. The dataset was riddled with inconsistencies, missing values, and formatting issues that made it unsuitable for analysis in its original state. Using Python and its powerful data manipulation library, pandas, I cleaned the dataset by handling missing values, correcting data types, and standardizing formats. This process involved a variety of techniques, including data imputation, outlier detection, and the use of regular expressions. Once the data was cleaned, I conducted exploratory data analysis to uncover trends and patterns in the data. I used matplotlib and seaborn for data visualization, creating a variety of plots to better understand the underlying patterns in the data. This project demonstrated my ability to handle real-world data cleaning tasks and to draw meaningful insights from a cleaned dataset. It showcased my proficiency in Python, pandas, and data visualization libraries, as well as my attention to detail and problem-solving skills in the face of messy data.

EDA Diagnosing Diabetes

In this project, I conducted an exploratory data analysis (EDA) on a diabetes dataset. The main goal was to identify patterns and relationships in the data that could aid in the diagnosis of diabetes. The dataset included various health metrics such as BMI (Body Mass Index) and Skin Thickness, among others. Python, along with libraries like pandas for data manipulation and matplotlib for data visualization, were used extensively throughout the project. One of the key visualizations created was a scatterplot of BMI vs Skin Thickness. This plot helped in understanding the relationship between these two variables, which could potentially be significant in diagnosing diabetes. The project demonstrated my ability to conduct thorough exploratory data analysis using Python and its powerful libraries. It also showcased my skills in creating meaningful data visualizations to understand complex datasets and draw insights from them.

Titanic Survival Analysis

In this project, I conducted an in-depth analysis of the Titanic dataset, focusing on the survival rates of passengers based on their embarkation point. The dataset included various passenger details, but the primary focus was on the 'Embarked' feature, which indicates the port of embarkation. The main tool used for this analysis was Python, with libraries including pandas for data manipulation, matplotlib for data visualization, and pywaffle for creating waffle charts. One of the key components of the project was the creation of a waffle chart to visualize the distribution of survivors based on their embarkation point. This provided valuable insights into the survival rates of passengers from different ports. The project demonstrated my ability to manipulate data using pandas, create meaningful visualizations with matplotlib and pywaffle, and draw insights from complex datasets.
